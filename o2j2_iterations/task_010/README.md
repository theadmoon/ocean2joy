## 🎯 КРАТКОЕ ОПИСАНИЕ ЗАДАЧИ

Navbar в O2J2 отличается от прототипа Ocean2Joy по 7 критическим параметрам:

1. ❌ Логотип — используется текст вместо SVG-файла.
2. ❌ Публичные ссылки меню — якорный формат `/#services` вместо `/services`.
3. ❌ Состояние "залогинен" — иконки Lucide вместо текста.
4. ❌ Quick Switch — отсутствует блок для переключения ролей.
5. ❌ Кнопка "Start Project" — ведёт на `/register` вместо `/request`.
6. ❌ Мобильное меню — якорные ссылки вместо прямых маршрутов.
7. ❌ Импорты — `lucide-react` вместо `react-icons/hi`.
