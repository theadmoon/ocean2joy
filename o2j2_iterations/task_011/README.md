## 🎯 КРАТКОЕ ОПИСАНИЕ ЗАДАЧИ

Hero Section в O2J2 отличается от прототипа Ocean2Joy по 5 критическим параметрам:

1. ❌ **Шрифт** — отсутствует определение `font-family` для `body` → текст выглядит тоньше.
2. ❌ **Цвет "Ocean"** — `text-yellow-400` вместо `text-yellow-300` → менее яркий контраст.
3. ❌ **Кнопка "Start Your Project"** — динамический маршрут вместо `/request`.
4. ❌ **Кнопка "Explore Services"** — якорь `#services` вместо `/services`.
5. ❌ **data-testid** — присутствуют (их нет в прототипе).
